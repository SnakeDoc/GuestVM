Introduction

This documentation describes the guest VM microkernel, GUK for short, which is an evolution of the Xen Minimal OS, Mini-OS. GUK was designed initially to support the Virtual edition of the Maxine virtual machine and, therefore, although more functional than Mini-OS, it is also somewhat more specialized. However, GUK should be usable in a wide variety of environments, particularly in the support of language run time virtual machines.

The main differences between Mini-OS, when originally forked, and GUK are:

  • Support for multiple VCPUS (SMP)
  • Pre-emptive thread scheduling
  • Memory management tailored to a language run time virtual machine
  • Memory ballooning
  • Block device support
  • Suspend/Resume support
Note that Mini-OS has also evolved since the fork and it would be good to merge bug fixes and new features as appropriate.
